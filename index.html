Project Experience

Raytracer

A C++ program built from the ground up that renders images based on a text description. Lighting follows a variation of the Phong Illumination Model. Features include shadows, transparency, reflection, light attenuation, and depth cueing.

TypeScript Shooter

A simple space shooter that uses TypeScript. Features music, sound effects, score/life tracking, and a boss phase. Live build available here.

Earthquake Visualizer

A TypeScript project that displays a series of earthquakes across the last 150 years on a static map or a rotating globe. Live build available here.

Perlin Noise Generator

A C++ program that takes in image dimensions, a color, and a desired frequency and generates a .ppm image using Perlin Noise. A complementary color is calculated from the given color.

Stone Soup

An Android application written in Java designed to facilitate easier planning of social meal events. Allows for scheduling of events, viewing recipes, and organizing who is coming and what each person will bring.

StudySpaces

An Android application written in Java that allows for the viewing and reserving of various study spaces that are available around the University of Limerick campus.

Maze Attack!

A game created using Unity where the user can design and build a maze with walls and traps for enemies to navigate through using an implementation of the A* search algorithm.

Drone Rescue Simulation

A C++ program that simulates a drone flying in automatic search patterns over a low-res 3D model of the UMN campus to look for a hidden robot using the Canny edge-detection algorithm.
